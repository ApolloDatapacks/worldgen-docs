## Chorus Plant

The `minecraft:chorus_plant` feature type is used to place chorus plants. In vanilla, they are placed in the End Highlands biomes.

### JSON format

```json
{
   "type": "minecraft:chorus_plant",
   "config": {}
}
```

This feature has no fields to configure.
